Checking the correlation matrix it appears that the target, 'charges', is highly correlated with 'smoker' and 'age'.
It is minimally correlated with 'children' and 'sex'. I will remove them due to the poor correlation. Also, 'children' is positively correlated with 'age' which may cause auto-correlation issues.
Likewise 'sex' is slightly correlated with 'smoker'. Again, the feature less correlated with the target will be removed to prevent auto-correlation issues.
Furthermore, 'region', correlates with 'bmi' which is correlated with the target. Consequently, 'region' will be removed.

This leaves the target feature plus three others: 'age', 'bmi' and 'smoker'.
The 'smoker' feature is categorical with levels of 'yes' and 'no'. Categorical features will cause problems with the linear regression model and will need to be dealt with.
The pandas library contains the 'get_dummies' method which will be used to encode the 'smoker' feature.

Encode categorical feature¶

The dataframe is passed as the data parameter which allows the method to determine which features to encode. The 'prefix' parameter is used to assign a value to the resulting columns for ease of readability. The original 'smoker' feature will be dropped and two new encoded features added. The prefix 'smoker' will be added to the new columns.
